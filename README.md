# Сайт-конструктор для онлайн обучения

## Структура проекта
- `server/` - Бэкенд (Node.js + Express + SQLite)
- `client/` - Фронтенд (React + Vite)

## Запуск проекта

### 1. Сервер
```bash
cd server
npm run dev
```
Сервер запустится на `http://localhost:5000`.

### 2. Клиент
```bash
cd client
npm run dev
```
Клиент запустится на `http://localhost:5173`.

## Стек
- **Frontend:** React, Tailwind CSS
- **Backend:** Node.js, Express
- **База данных:** SQLite
