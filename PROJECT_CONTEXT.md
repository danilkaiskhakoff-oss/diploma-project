## Технологический стек
- **Frontend:** React 18 + Vite + Tailwind CSS + Framer Motion
- **Backend/BaaS:** Firebase (Firestore, Authentication)
- **Хостинг:** Render.com (SPA)
- **Сборщик:** Vite (hot module replacement, ES modules)
- **Анимации:** Framer Motion
- **Язык:** JavaScript (ES2020+)

## Архитектура проекта
Проект — SPA (Single Page Application) с условным рендерингом маршрутов:
- Главная страница → выбор уровня сложности
- 3D-карта уровней с интерактивными чекпоинтами
- Симуляции (14 штук) — интерактивные сценарии
- Квизы — проверка знаний после каждого модуля
- Брифинги — теоретический материал для Advanced уровней
- Админ-панель (/admin) — управление контентом через Firebase

## Ключевые компоненты

### Основные экраны
- `App.jsx` — корневой компонент, управление состоянием (selectedLevel, isAdminRoute)
- `LevelSelect.jsx` — выбор уровня сложности (анимированные карточки)
- `Map3D.jsx` — 3D-карта с интерактивными чекпоинтами
- `CheckpointScene.jsx` — контейнер для загрузки симуляций/квизов

### Симуляции (14 штук)
1. `CyberBasicsSimulation.jsx` — рабочий стол Windows 7, основы
2. `PhishingStage.jsx` — анализ email на фишинг
3. `DetectionTriage.jsx` — классификация алертов в SOC
4. `DDoSSimulation.jsx` — отражение DDoS-атаки (NOC Dashboard)
5. `PentestSimulation.jsx` — сканирование сети (Nmap)
6. `OSINTSimulation.jsx` — разведка (Google Dorks, Shodan)
7. `MalwareSimulation.jsx` — классификация малвари
8. `WifiSecuritySimulation.jsx` — защита Wi-Fi
9. `NetworkAttacksSimulation.jsx` — MITM, DNS Spoofing
10. `SocialEngineeringSimulation.jsx` — социальная инженерия
11. `SocialMediaSimulation.jsx` — безопасность в соцсетях
12. `PasswordSimulation.jsx` — безопасность паролей
13. `DataProtectionSimulation.jsx` — защита данных
14. `InsiderThreatsSimulation.jsx` — внутренние угрозы

### Админ-панель
- `AdminPanel.jsx` — корневой компонент админки
- `AdminLogin.jsx` — форма входа (Firebase Auth)
- `AdminDashboard.jsx` — дашборд со статистикой
- `QuizEditor.jsx` — редактор квизов
- `LevelEditor.jsx` — редактор уровней
- `BriefingEditor.jsx` — редактор брифингов
- `ThemeEditor.jsx` — редактор темы
- `AdminSidebar.jsx` — навигация

### Данные (Firestore коллекции)
- `levels` — 3 документа (beginner, intermediate, advanced), каждый с массивом checkpoints
- `quizzes` — 14 документов с вопросами
- `briefings` — 4 документа (ddos, pentest, incident-response, osint)
- `ui-config` — 1 документ (global) с настройками темы

## База данных (Firebase Firestore)
- Облачная NoSQL база
- Real-time обновления
- Security Rules: чтение — всем, запись — только авторизованным админам
- Fallback на локальные данные (JS-файлы) при отсутствии Firebase

## Аутентификация

- Регистрация отключена, админы добавляются вручную
